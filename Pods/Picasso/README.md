A view for rendering CIImage with Metal/OpenGL.

Picasso is [YUCIImageView](https://github.com/YuAo/YUCIImageView) in Swift.


## Requirements

iOS 8.0

Swift 3.0

## Installation

### CocoaPods

```ruby
pod 'Picasso', '~> 0.4.1'
```

### Carthage

```swift
github "Limon-O-O/Picasso"
```
